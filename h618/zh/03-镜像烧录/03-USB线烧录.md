# 03-USB线烧录

## 1. Android USB线 镜像烧录

镜像可通过网盘这个路径下载

``` 
h618_data\2-Image\K2*\Android12.0_*
```



### 准备工具

**下载烧录工具**

```
h618_data\5-DevelopmentTools\USB_ImageBurnTool\PhoenixSuit.zip
```

**硬件准备工具**

准备一根 USB A to C 烧录线，用于连接烧录主机 于 开发板



**主板进入烧录模式方法**

先按住板子背面的MASKROM按键 

再连接TYPE-C口和电脑

电脑设备管理器有识别设备即可
