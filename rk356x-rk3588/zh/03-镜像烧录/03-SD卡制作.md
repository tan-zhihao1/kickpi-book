# 03-SD卡制作

此文档用于指导如何制作SD启动卡和烧录卡



## SD卡制作工具

### 安装SDDiskTool

**网盘路径：**

```
rk356x-rk3588_data\5-DevelopmentTools\win_x64_SDDiskTool\SDDiskTool_v1.74.zip
```

下载后解压即可使用

### 语言选择

语言默认中文，可以通过解压后SDDiskTool_v1.74目录下config.ini修改

```
[Language]
Kinds=2
Selected=1
LangPath=Language\
```

> Selected=1(中文);Selected=2(英文)
